Welcome to GitHub Page for the Hacking Health Camp Hackathon project of team #26 Medical Graphs.

This project contains all the technical resources and code of tests, prototypes and demo of the medical graph project done and used during the Hackathon.

Most of the work done are HTML pages using Vis.js JavaScrit library to display graphs and can be accessed directly here:

  • Demo: Application shown during presentation following a consultation scenario.
  • Technical prototype: Prototype displaying a graph with different icons and styles to evaluate the Vis.js graph library on the subject.
  • Display Graph: example code to retrieve the content of a Neo4j database and display it with Vis.js.
  • Popoto.js: example of Popoto.js customization on the Neo4j movie example database using Vis.js to display results as graph.
